# Tail Match — ロードマップ

## 現在地（2025年11月時点）

基盤は完成しています。フロントエンド・データベース・スクレイパーが動いており、20都道府県・34施設の情報を収集・表示できています。管理画面と統計ダッシュボードも稼働中です。

ここから「全国のすべての保護猫情報を網羅する」という目標に向けて、段階的に拡張していきます。

---

## 進行中

### 全国55施設への対応拡張

残り27都府県・55施設のスクレイパーを順次実装します。

優先順位は以下の通りです：

1. **政令指定都市**（新潟、静岡、浜松、名古屋、京都、岡山、北九州、福岡、熊本）
2. **県庁所在地**（36都市）
3. その他の中核施設

施設ごとにサイト構造が異なるため、一律の対応ではなく個別に実装しています。特に難しいのは堺市・横浜市のように、情報が画像で提供されているケースです。これらには OCR 統合も視野に入れています。

---

## 次の実装候補

### Instagram 風のビューア

写真を主役にした、縦スクロール型の閲覧体験を追加予定です。

現在のリスト表示とは別に、写真を大きく見せるビューを用意することで、猫の個性をより伝えやすくします。

### 前日からの変化表示

昨日と比べて新しく加わった猫・譲渡が決まった猫を強調表示する機能です。毎日チェックしているユーザーが「今日の新着」をすぐ把握できるようにします。

### 通知機能

条件を保存しておき、新しい猫が登録されたら通知を受け取れる仕組みです。「白猫が来たら教えてほしい」という使い方を想定しています。ブラウザ通知またはメール通知を検討しています。

### お気に入り登録

気になる猫をブックマークする機能です。ログイン不要で使えるローカル保存を基本とし、アカウント登録でデバイスをまたいで共有できる形も検討します。

## 長期的な方向性

### 全国完全網羅

47都道府県すべての公的保護施設をカバーすること。これが最も重要な目標です。どんなに UI が良くても、情報が揃っていなければ役に立ちません。

### 民間シェルター・NPOへの対応

自治体施設だけでなく、民間の保護団体やNPOの情報も取り込むことを検討しています。ただし情報の信頼性確保が課題です。

### データの公開

収集した保護猫データを API 公開することで、他の開発者が活用できる基盤にすることも視野に入れています。

---

## やらないこと

- 仲介・斡旋業務（情報提供に徹する）
- 会員制・有料プラン（無料で使えることが前提）
- 猫以外の動物への拡張（猫専門を維持する）
