# Tail Match — サービス概要

## 何を作っているか

全国の保護猫情報を一か所で探せる、猫専門のマッチングサービスです。

日本には各都道府県・市区町村の動物愛護センターや保護施設が存在しますが、それぞれが独自のウェブサイトで情報を公開しており、横断的に検索する手段がありませんでした。「近くのシェルターにどんな猫がいるか」「白猫を探している」といった当たり前のニーズに、当たり前の答えが返ってこない状態が続いていました。

Tail Match はその空白を埋めます。各施設のサイトを自動で巡回して猫の情報を収集し、統一されたインターフェースで検索・閲覧できるようにしたサービスです。

## なぜ猫専門なのか

犬と猫を混在させた動物マッチングサービスは既にいくつか存在します。しかし「猫だけを専門に扱う、全国規模のサービス」は存在しませんでした。

人間用の求人サービスや部屋探しサービスが職種・エリア別に細分化されているように、猫を探す人と猫を救いたい人のためだけに最適化されたサービスがあっていい——というのがこのプロジェクトの出発点です。

「世にないものを作る」がモットーです。

## 既存サービスとの差別化

| 観点 | 既存サービス | Tail Match |
|------|-------------|------------|
| 対象動物 | 犬・猫・その他を混在 | 猫のみに特化 |
| 情報源 | 団体が手動で登録 | 自治体サイトを自動収集 |
| カバレッジ | 登録した団体のみ | 全国の公的施設を網羅予定 |
| 更新頻度 | 団体任せ | 定期的な自動巡回 |
| 緊急度表示 | なし or 単純 | 視覚的なアラート表示 |

## 解決したい問題

- 保護猫の情報が施設ごとに孤立していて、探す人が見つけられない
- 殺処分期限が近い猫が、遠くにいる里親候補に届かない
- 施設側が情報発信に割けるリソースが限られている

情報を集めて届ける部分を自動化することで、人が関われる部分——実際に迎える判断と行動——に集中できる環境を作ります。

## 現在地

2025年11月時点で、20都道府県・34施設の情報収集に対応しています。全国47都道府県への完全対応に向けて継続拡張中です。

サービスは https://tail-match.llll-ll.com で公開中です。
