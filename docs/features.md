# Tail Match — 機能一覧

## シェルター情報の自動収集

各自治体の動物愛護センター・保護施設のウェブサイトを定期的に巡回し、保護猫の情報を自動取得します。

サイトの構造はバラバラです。テーブル形式のところもあれば、カード形式のところも、PDF掲載のところもあります。JavaScript で動的にコンテンツを生成するサイトにも対応しています。施設ごとに専用の収集ロジックを実装することで、できるだけ多くの情報を正確に拾い上げます。

収集は礼儀正しく行います。アクセス間隔を空け、robots.txt を尊重します。

現在の対応状況：

- 対応済み：20都道府県、34施設
- 実装済みスクレイパー数：50
- 全国目標：47都道府県（78施設）

## 猫の検索とフィルタリング

収集した猫の情報を、以下の条件で絞り込めます：

- **地域**：都道府県・施設で絞る
- **性別**：オス・メス・不明
- **年齢**：子猫・成猫・シニア
- **緊急度**：期限が近い順に優先表示
- **性格・特徴**：人懐っこい・穏やか・活発などの性格特徴で絞る
- **キーワード**：名前・品種・毛色・性格をテキスト検索

複数条件を組み合わせて検索できます。「神奈川県のシェルターにいる女の子の子猫」といった絞り込みが可能です。

検索条件はURLに反映されるため、結果をブックマークしたりSNSで共有できます。

## 緊急度の表示

保護期間の期限が近い猫を視覚的に強調します。色とアニメーションで緊急度を伝え、見落としを防ぎます。

期限が差し迫った猫が、サイトを開いた人の目に止まる確率を上げることが目的です。

## 統計ダッシュボード

現在の保護猫情報を集計して表示します：

- 全国の保護猫総数
- 地域別の分布
- 緊急度別の内訳（緊急・要注意・注意の3段階）
- 施設ごとの掲載数

すべての統計数値はクリック可能で、該当する検索結果に直接飛べます。地域別の分布も都道府県ごとにフィルタされた検索ページにリンクしています。

## 管理画面（運用者向け）

自動収集システムの動作状況を確認できます：

- スクレイピングの実行履歴（タイムライン表示）
- 施設ごとの成功率・実行時間・発見数
- エラーや警告の確認
- 前回から増減した猫の数の確認

収集した匹数が前回より減った場合は自動で警告を出します。データの品質を維持するための仕組みです。

## 猫の詳細ページ

個別の猫ごとに詳細ページがあります：

- 名前・性別・年齢・特徴などの基本情報
- 掲載施設の連絡先情報
- 元サイトへのリンク

里親希望の問い合わせは施設に直接行います。Tail Match は情報の橋渡しに徹し、仲介手数料などは発生しません。

## トップページの緊急・要注意セクション

トップページには期限が迫った猫を2段階で表示します：

- **緊急のシッポたち**：残り3日以内の猫をサムネイルで一覧表示
- **要注意のシッポたち**：残り3〜7日の猫をサムネイルで一覧表示

見過ごしを防ぎ、急を要する猫に注意を向けやすくしています。

## その他の機能

- **施設一覧**：全国の対応施設を一覧で確認できます
- **宇宙猫ゲーム**：サイト内のちょっとした息抜き要素
- **法的ページ**：プライバシーポリシー・利用規約・免責事項を整備済み
- **モバイル対応**：スマートフォンからも快適に使えます
- **favicon**：ブラウザタブやブックマークにロゴの「て」が表示されます
