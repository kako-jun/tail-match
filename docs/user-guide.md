# Tail Match — 使い方ガイド

## 猫を探す

### トップページから始める

サイトを開くと、現在登録されている保護猫の写真がカルーセル形式で流れます。気になる子がいれば、そのままクリックして詳細を確認できます。

### 条件で絞り込む

画面上部の検索・フィルタエリアで条件を指定します。

**地域で絞る**
都道府県または施設名を選びます。「近所のシェルターだけ見たい」「特定の県の情報を集めたい」といった使い方ができます。

**特徴で絞る**
性別・年齢などの条件を追加できます。複数の条件を組み合わせることも可能です。

**緊急度で絞る**
保護期間の期限が近い猫を優先して表示できます。急いで里親を探している猫を見逃したくない場合に使ってください。

### 検索結果を見る

条件に合う猫がカード形式で一覧表示されます。緊急度が高い猫は視覚的に強調されています。

気になる猫をクリックすると詳細ページに移動します。

## 猫の詳細を確認する

詳細ページには以下の情報が掲載されています：

- 名前（ある場合）
- 性別・推定年齢
- 体の特徴や性格（施設が公開している情報）
- 掲載施設の名前・所在地・連絡先

「元サイトを見る」リンクから、施設の原文ページに直接アクセスできます。写真が多い場合や、より詳しい情報が必要な場合は元サイトを参照してください。

## シェルター情報を見る

ナビゲーションの「施設一覧」から、対応している全施設の情報を確認できます。

施設ごとに以下の情報があります：
- 施設名・所在地
- 電話番号・ウェブサイトURL
- 現在の掲載猫数

気になる施設があれば、直接電話や見学の問い合わせができます。

## 里親になるには

Tail Match は情報提供サービスです。里親申込みの手続きは、各施設に直接問い合わせてください。

施設によって手続きが異なります。多くの場合：
1. 施設に電話または来館して問い合わせ
2. 面談・審査
3. トライアル期間
4. 正式譲渡

詳細は各施設のウェブサイトや電話でご確認ください。

## お気に入り（今後対応予定）

気になる猫をブックマークしておく機能は現在開発中です。ログインなしで使えるローカル保存の仕組みを検討しています。

## 情報の鮮度について

掲載情報は定期的な自動収集で更新されます。ただし、施設側で情報が更新されてから Tail Match に反映されるまでタイムラグがある場合があります。

「この猫はまだいますか？」という確認は、施設に直接問い合わせるのが確実です。
