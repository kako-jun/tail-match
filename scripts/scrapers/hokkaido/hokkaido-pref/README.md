# 北海道立動物愛護センター「あいにきた」スクレイパー

## 情報

- **URL**: https://www.pref.hokkaido.lg.jp/ks/awc/inuneko.html
- **Municipality ID**: 19
- **データタイプ**: 譲渡動物（adoption）
- **対象**: 新しい飼い主を募集中の猫

## 特徴

- **h3とh4タグ混在**: 猫によってヘッディングタグが異なる
- **詳細な猫情報**: 名前、性格、健康状態の詳細な説明
- **複数画像対応**: 各猫につき1〜2枚の画像

## 使用方法

```bash
node scripts/scrapers/hokkaido/hokkaido-pref/scrape.js
node scripts/scrapers/hokkaido/hokkaido-pref/html-to-yaml.js
```

## HTMLパース詳細

### 見出し形式

- h3またはh4タグ: `（仮名）ほぅ（雑種ギジトラ メス 成猫）`
- パターン: `（仮名）{名前}（雑種{毛色} {性別} {年齢}）`

### 抽出フィールド

- 名前: （仮名）の後の文字列
- 毛色: 雑種の後の文字列
- 性別: オス/メス
- 年齢: 成猫、推定〜歳、幼猫など
- 画像: 次の見出しまでのimg要素

## 注意事項

- 治療中の猫も含まれる
- 譲渡交渉中の猫も掲載されている
- h3とh4タグの両方をチェックする必要がある
