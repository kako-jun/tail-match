# 札幌市動物愛護管理センター「あいまる さっぽろ」スクレイパー

## 情報

- **URL**: https://www.city.sapporo.jp/inuneko/syuuyou_doubutsu/jotoneko.html
- **Municipality ID**: 20
- **データタイプ**: 譲渡動物（adoption）
- **対象**: 譲渡可能猫情報

## 特徴

- **2つの形式**: 子猫（テーブル形式）と成猫（h3タグ形式）
- **詳細な猫情報**: 名前、性格、健康状態の説明付き
- **画像付き**: 成猫は1〜2枚の画像あり

## 使用方法

```bash
node scripts/scrapers/hokkaido/sapporo-city/scrape.js
node scripts/scrapers/hokkaido/sapporo-city/html-to-yaml.js
```

## HTMLパース詳細

### 子猫（テーブル形式）

- 収容番号、性別、色、推定月齢の4列構造
- 例: LC071027-03, オス, 黒, ４週

### 成猫（h3タグ形式）

- パターン: `{名前}（{収容番号}） {年齢} {性別}`
- 例: `レム（AC071107-01） 5歳 メス`
- 次の要素に画像と説明文

### 抽出フィールド

- 子猫: 収容番号、性別、毛色、推定月齢
- 成猫: 名前、収容番号、性別、年齢、画像

## 注意事項

- 子猫は写真掲載なし（体調不安定のため）
- 成猫は「人に馴れている猫」と「あまり人に馴れていない猫」に分類
- 収容番号は名前としても使用（子猫の場合）
